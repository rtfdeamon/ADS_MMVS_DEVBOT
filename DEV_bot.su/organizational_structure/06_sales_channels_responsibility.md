- Работа с семантическим ядром

- Мониторинг упоминаний и обратной связи
- **Отчетность:** Еженедельный отчет по публикациям и трафику

✅ **Telegram (Ответственный)**
- Составление списка релевантных каналов и чатов
- Подготовка нативных сообщений
- Участие в обсуждениях
- Мониторинг реакции аудитории

- Ведется учет качества лидов и проектов по каналам

### 2. ПЕРЕГОВОРЩИК

#### Зона ответственности: Продажи и работа с клиентами

**Каналы продаж:**

✅ **Все каналы привлечения (Обработка лидов)**
- Получение лидов от маркетолога
- Квалификация лидов (соответствие ЦА)
- Первичный контакт с потенциальными клиентами
- Ведение переговоров
- Закрытие сделок
- **Отчетность:** Ежедневный отчет по лидам и сделкам

✅ **Проекты из чатов аутстаффа (Обработка проектов)**
- Получение проектов из чатов аутстаффа от маркетолога
- Анализ требований проекта
- Оценка соответствия возможностям команды
- Ведение переговоров по проектам
- Подготовка коммерческих предложений
- Закрытие сделок по аутстафф проектам
- **Отчетность:** Ежедневный отчет по проектам и сделкам

✅ **Конференции (Участник)**
- Участие в мероприятиях
- Прямые продажи на конференциях
- Сбор контактов
- Последующая работа с контактами
- **Отчетность:** По событию (список контактов, результаты)

✅ **Партнерские каналы (Ответственный)**
- Поиск и привлечение партнеров
- Ведение переговоров с партнерами
- Управление партнерскими отношениями
- **Отчетность:** Ежемесячный отчет по партнерам

**Каналы мониторинга:**

✅ **CRM (Ответственный)**
- Ведение воронки продаж в CRM
- Отслеживание статусов сделок
- Заполнение карточек клиентов
- Анализ конверсий по этапам воронки
- **Отчетность:** Ежедневный отчет по воронке, еженедельный анализ

**Обратная связь маркетологу:**
- Информирование о качестве лидов по каналам
- Статистика конверсии лидов в сделки по источникам
- Рекомендации по оптимизации каналов
- **Отчетность:** Еженедельный отчет по качеству лидов

## Процессы взаимодействия

### Ежедневный процесс работы с лидами

1. **Маркетолог** собирает лиды из всех каналов
2. **Маркетолог** передает лиды **Переговорщику** в течение 1 часа
3. **Переговорщик** обрабатывает лиды в течение рабочего дня
4. **Переговорщик** ведет учет в CRM
5. **Переговорщик** информирует **Маркетолога** о качестве лидов

### Еженедельный процесс отчетности

1. **Понедельник, до 12:00:**
   - **Маркетолог** отправляет еженедельный отчет
   - **Переговорщик** отправляет еженедельный отчет

2. **Понедельник, 14:00:**
   - Совещание с генеральным директором
   - Обсуждение отчетов
   - Планирование недели

### Ежемесячный процесс отчетности

1. **5 число месяца:**
   - **Маркетолог** отправляет ежемесячный отчет
   - **Переговорщик** отправляет ежемесячный отчет

2. **Первая неделя месяца:**
   - Совещание с генеральным директором
   - Анализ результатов месяца
   - Планирование следующего месяца

## Эскалация проблем

### Уровень 1: Операционные проблемы
- Решаются между маркетологом и переговорщиком
- Пример: качество лидов, скорость обработки

### Уровень 2: Тактические проблемы
- Решаются на еженедельном совещании
- Пример: невыполнение плана по лидам, проблемы с каналами

### Уровень 3: Стратегические проблемы
- Эскалируются генеральному директору
- Пример: невыполнение плана продаж, проблемы с бюджетом
