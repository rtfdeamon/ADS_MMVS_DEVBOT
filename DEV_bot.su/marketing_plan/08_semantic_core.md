# 3.1. Семантическое ядро (для услуг по заказной разработке)

**Цель:** Предоставить вам готовый набор ключевых слов для запуска первой рекламной кампании в Яндекс.Директ. Эти слова сгруппированы по потребностям аудитории, ищущей подрядчика на AI-разработку.

**Инструкция:**
*   Используйте одну группу слов на одну группу объявлений в Яндекс.Директ.
*   Перед использованием проверьте релевантность слов вашему конкретному предложению.
*   Не забудьте добавить минус-слова, чтобы отсечь нецелевой трафик.

**Базовые минус-слова (добавить на уровне кампании):**
`-бесплатно, -обучение, -курсы, -вакансия, -своими, -руками, -что, -это, -реферат, -скачать, -примеры, -как, -статья, -обзор, -книги, -готовые, -платформа, -сервис`

---
### Группа 1: Прямой спрос на разработку (самые целевые)

*Для тех, кто ищет подрядчика на разработку.*

```
заказать разработку ai
разработка ии на заказ
ai разработка для бизнеса
заказная разработка ai
разработка ml моделей на заказ
найти ai разработчиков
команда ai разработчиков
студия ai разработки
разработка нейросетей на заказ
```

---

### Группа 2: Нишевый спрос на разработку (средне-целевые)

*Для тех, кто ищет разработку под конкретную отрасль.*

```
ai разработка для e-commerce
ии для ритейла заказать
разработка ai для логистики
кастомные ai решения для ритейла
разработка чат-ботов для интернет-магазина
создание системы прогнозирования спроса
```

---

### Группа 3: Спрос на экспертизу (широкие)

*Для тех, кто ищет консультацию или экспертизу, но может стать клиентом на разработку.*

```
ai для оптимизации логистики
ии для клиентского сервиса в e-commerce
как внедрить ai в интернет-магазин
ai для снижения возвратов
автоматизация склада с помощью ai
консультации по внедрению ии
```

---

### Группа 4: Конкурентные запросы (для РСЯ)

*Для показа тем, кто ищет ваших конкурентов (студии, интеграторов).*

```
(названия конкурирующих студий и интеграторов)
заказать проект у [конкурент]
[конкурент] отзывы
альтернатива [конкурент]
```
